DocPlanner PHP SKD
==================

Requirements
------------

 * PHP >= 5.4
 * guzzle/guzzle >= 3.3
 * guzzle/plugin-oauth >= 3.3
 * symfony/http-foundation >= 2.3

Installation
------------

```bash
composer require 'lbarulski/docplanner-sdk:*'
```

Example
-------

{% highlight php %}
<?php

use DocPlanner\SDK\DocPlannerSDK;

$dp = new DocPlannerSDK('ConsumerKey', 'ConsumerSecret');

$result = $dp->doctor()->categories();
var_dump($result);
{% endhighlight %}
